# RxzTheme 主题

<rxz-theme theme="th-red">
  <rxz-button type="primary">主要按钮</rxz-button>
</rxz-theme>

``` vue
<template>
  <rxz-theme theme="th-red">
    <rxz-button type="primary">主要按钮</rxz-button>
  </rxz-theme>
</template>
```

## Attribute 属性

| 参数    | 类型    | 描述           | 可选值     | 默认值     | 必须 |
| ------- | ------- | -------------- | ---------- | ---------- | ---- |
| theme    | String  | 主题       | default/th-red         | default    |      |
