# rxz-overflow 溢出指令

<div style="width: 100px" v-rxz-overflow>  
  我的名字叫rxz-overflow
</div>

``` vue
<div style="width: 100px" v-rxz-overflow>  
  我的名字叫rxz-overflow
</div>
```

## Attribute 属性

| 参数       | 类型    | 描述                 | 可选值     | 默认值 | 必须 |
| ---------- | ------- | -------------------- | ---------- | ------ | ---- |
| isScroll   | Boolean | 超出最大高度是否滚动 | true/false | true   |      |
| showShadow | Boolean | 是否显示阴影         | true/false | false  |      |
