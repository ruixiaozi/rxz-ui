# 排序

## sort 排序（使用快速排序）

```
function sort(arr,compare)
```

+ 参数

  + arr：Array 待排序数组

  + compare：Function 比较方法（可选）

    参数：a、b

    结果：a>b返回正数，a==b返回0，a<b返回负数

+ 返回值

  ​	排序好的数组
